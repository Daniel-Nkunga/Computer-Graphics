## Rendering
3. **Object based rendering**: pipeline of task happen to every object in the scene in order to render them
	1. **Project**: convert 3D environment onto a 2D shape
	2. **Clip**: determine/hide all non-visible polygons
	3. **Rasterize**: convert polygon data to pixels
	4. **Shade**: color pixels properly

## History of Graphics Engines
- Other (historical) graphics library:
	- GKS (Graphical Kernel System) - only supports 2D; meant more for business applications
	- Core
	- GKS 3D
	- PHIGS (Programmers Hierarchical Graphics System)
	- X Windowns System
	- PEX
- Silicon Graphics (SGI) revolutionized graphics in the 1980s by designing hardware that directly interfaced with the GPU
	- Propriety; called GL
- Success of GL lead to the development of OpenGL in the 1990s focused on avoiding OS dependencies and run by multiple companies

### Example 1: White square on screen
```cpp
#include <GL/glut.h>
void mydisplay()
{
	glClear(GL_COLOR_BUFFER_BIT);
	glBegin(GL_POLYGON);
	glVertex2f(-0.5, -0.5);
	glVertex2f(-0.5, 0.5);
	glVertex2f(0.5, 0.5);
	glVertex2f(0.5, -0.5);
	glEnd(); //End the polygon
	glFlush();
	// Defualt color is WHITE; default coordinate goes from X:[-1,0, 1,0] and Y[-1.0, 1.0]
}

int main(int argc, char** argv)
{
	glutCreateWindow("simple"); // Create window of default size and default location on screen (depending on system); you pass in name
	glutDisplayFunc(mydisplay); // Callback function of function defined above
	glutMainLoop(); // Forces an infinite loop
}
```
